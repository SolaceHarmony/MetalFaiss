Create a view into the array with the given shape and strides.

The resulting array will always be as if the provided array was row
contiguous regardless of the provided arrays storage order and current
strides.

<div class="admonition note">

Note

Note that this function should be used with caution as it changes the
shape and strides of the array directly. This can lead to the resulting
array pointing to invalid memory locations which can result into
crashes.

</div>
