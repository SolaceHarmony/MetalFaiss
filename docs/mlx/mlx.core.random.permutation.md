## Curated Notes

- If `x` is an integer n, returns a permutation of 0..n-1; if `x` is an array, permutes entries along `axis`.
- For reproducibility, pass `key=`; there is no `device=` argument.

### Examples

```python
import mlx.core as mx

p = mx.random.permutation(10)              # permutation of 0..9

x = mx.arange(12).reshape(3, 4)
idx = mx.random.permutation(4)
xp = x[:, idx]                             # equivalent to axis=1 permutation

# Deterministic
k = mx.random.key(0)
pd = mx.random.permutation(10, key=k)
```
