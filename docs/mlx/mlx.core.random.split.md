## Curated Notes

- Use `mx.random.split(key, num)` to get `num` subkeys that produce independent streams.
- Reuse subkeys deterministically; avoid reusing the same key when you want different draws.
- No `device=`; placement follows default device or per‑op `stream`.

### Examples

```python
import mlx.core as mx

master = mx.random.key(0)
k1, k2 = mx.random.split(master, num=2)
a = mx.random.normal(shape=(4,), key=k1)
b = mx.random.normal(shape=(4,), key=k2)
```
