# mlx.nn.GroupNorm<a href="https://ml-explore.github.io/mlx/build/html/#mlx-nn-groupnorm"

class="headerlink" title="Link to this definition">#</a>  
Applies Group Normalization \[1\] to the inputs.

Computes the same normalization as layer norm, namely

<div class="math notranslate nohighlight">

\\y = \frac{x - E\[x\]}{\sqrt{Var\[x\]} + \epsilon} \gamma + \beta,\\

</div>

where <span class="math notranslate nohighlight">\\\gamma\\</span> and
<span class="math notranslate nohighlight">\\\beta\\</span> are learned
per feature dimension parameters initialized at 1 and 0 respectively.
However, the mean and variance are computed over the spatial dimensions
and each group of features. In particular, the input is split into
num_groups across the feature dimension.

The feature dimension is assumed to be the last dimension and the
dimensions that precede it (except the first) are considered the spatial
dimensions.

\[1\]: <a href="https://arxiv.org/abs/1803.08494"
class="reference external">https://arxiv.org/abs/1803.08494</a>

Parameters<span class="colon">:</span>  
- **num_groups**
  (<a href="https://docs.python.org/3/library/functions.html#int"
  class="reference external" title="(in Python v3.13)"><em>int</em></a>)
  – Number of groups to separate the features into

- **dims**
  (<a href="https://docs.python.org/3/library/functions.html#int"
  class="reference external" title="(in Python v3.13)"><em>int</em></a>)
  – The feature dimensions of the input to normalize over

- **eps**
  (<a href="https://docs.python.org/3/library/functions.html#float"
  class="reference external" title="(in Python v3.13)"><em>float</em></a>)
  – A small additive constant for numerical stability

- **affine**
  (<a href="https://docs.python.org/3/library/functions.html#bool"
  class="reference external" title="(in Python v3.13)"><em>bool</em></a>)
  – If True learn an affine transform to apply after the normalization.

- **pytorch_compatible**
  (<a href="https://docs.python.org/3/library/functions.html#bool"
  class="reference external" title="(in Python v3.13)"><em>bool</em></a>)
  – If True perform the group normalization in the same order/grouping
  as PyTorch.
