Implements sinusoidal positional encoding.

For more details see the paper
<a href="https://arxiv.org/abs/1706.03762"
class="reference external">Attention Is All You Need</a>.

Parameters<span class="colon">:</span>  
- **dims**
  (<a href="https://docs.python.org/3/library/functions.html#int"
  class="reference external" title="(in Python v3.13)"><em>int</em></a>)
  – The dimensionality of the resulting positional embeddings.

- **min_freq**
  (<a href="https://docs.python.org/3/library/functions.html#float"
  class="reference external" title="(in Python v3.13)"><em>float</em></a>*,*
  *optional*) – The minimum frequency expected. Default:
  <span class="pre">`0.0001`</span>.

- **max_freq**
  (<a href="https://docs.python.org/3/library/functions.html#float"
  class="reference external" title="(in Python v3.13)"><em>float</em></a>*,*
  *optional*) – The maximum frequency expected. Default:
  <span class="pre">`1`</span>.

- **scale**
  (<a href="https://docs.python.org/3/library/functions.html#float"
  class="reference external" title="(in Python v3.13)"><em>float</em></a>*,*
  *optional*) – A multiplicative scale for the embeddings. Default:
  <span class="pre">`sqrt(2/dims)`</span>.

- **cos_first**
  (<a href="https://docs.python.org/3/library/functions.html#bool"
  class="reference external" title="(in Python v3.13)"><em>bool</em></a>*,*
  *optional*) – If <span class="pre">`True`</span> embed using
  <span class="pre">`[cos(x);`</span>` `<span class="pre">`sin(x)]`</span>
  instead of the reverse. Default: <span class="pre">`False`</span>.

- **full_turns**
  (<a href="https://docs.python.org/3/library/functions.html#bool"
  class="reference external" title="(in Python v3.13)"><em>bool</em></a>*,*
  *optional*) – If <span class="pre">`True`</span> multiply the
  frequencies with
  <span class="math notranslate nohighlight">\\2\pi\\</span>. Default:
  <span class="pre">`False`</span>.
